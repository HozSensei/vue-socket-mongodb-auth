# Vue3 / Socket.io / MongoDB / Auth0

## Project setup
In each folder (Front-end & Back-end) launch the command : 
```
npm install
```

### Launch Front-end
```
npm run serve
```

### Launch Back-end
```
nodemon server.js
```
