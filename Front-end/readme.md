# Vue3 / Socket.io / MongoDB / Auth0

## Project setup
In each folder (Front-end & Back-end) launch the command : 
```
npm install
```
Then you need to edit configuration :  
in Back-end/server?js:7 (MongoDB url)  
And Front-End/src/main.js:12 (Auth0 informations)  

### Launch Front-end
```
npm run serve
```

### Launch Back-end
```
nodemon server.js
```
